# Live Event Prediction Game

Прототип игры, в которой пользователь смотрит трансляцию соревнования и пытается максимально точно предсказать момент игрового события.

## Идея
Игрок следит за ходом демо-матча и нажимает кнопку прогноза перед ожидаемым событием.  
Если событие действительно происходит в ближайшие секунды, игрок получает очки.  
Очки влияют на положение в турнирной таблице.

## Реализовано
- вход по нику
- запуск демо-матча
- таймер трансляции
- предсказание события пользователем
- автоматическое наступление событий в заданные моменты времени
- начисление очков за точный прогноз
- турнирная таблица
- лента событий матча

## Технологии
